# A股智能选股分析平台 PRD

## 1. 项目概述

**项目名称**: A股智选 - AI驱动的股票分析与选股辅助平台

**项目类型**: Web应用 (B/S架构)

**核心价值**: 利用大模型能力，帮助散户投资者进行行业趋势分析、把握市场热点、提供投资决策参考。

**目标用户**: A股个人投资者、散户

---

## 2. 功能需求

### 2.1 行业趋势分析

**功能描述**: 
- 一级行业（如同花顺行业分类：金融、地产、消费、科技等）走势分析
- 二级行业（细分行业）下的个股涨跌情况
- 资金行业内的流向、涨跌幅排名

**数据维度**:
- 行业指数涨跌幅
- 行业成交额
- 行业内个股涨跌幅分布
- 行业资金净流入/流出

**交互方式**:
- 树形结构展示行业层级
- 点击展开/收起二级行业
- 点击行业可查看详情面板

### 2.2 大盘星图可视化

**功能描述**: 
参考东方财富网大盘星图，以散点图形式展示A股全市场股票

**可视化规则**:
- **气泡面积**: 代表市值（流通市值/总市值）或成交额，可切换
- **气泡颜色**: 代表涨跌幅（红色=上涨，绿色=下跌，颜色深浅=涨跌幅大小）
- **时间周期**: 可选 3日/5日/10日/20日/60日/年初至今
- **市场筛选**: 沪深A股/上证/深证/创业板/科创板/北证

**交互功能**:
- 鼠标悬停显示股票基本信息（代码、名称、涨跌幅、市值）
- 双击跳转股票详情页
- 支持我的自选股高亮显示
- 支持全屏模式

### 2.3 市场热点分析

**功能描述**: 
- 实时从财经社区（雪球、同花顺股吧、东方财富股吧等）抓取热门讨论帖子
- 利用大模型分析帖子情感/热点话题
- 可视化展示热点板块、热门股票

**数据来源**:
- 雪球热门帖子
- 东方财富股吧
- 同花顺热帖

**分析维度**:
- 热门股票TOP30
- 热门话题/关键词
- 情感分析（看涨/看跌/中性）
- 帖子热度趋势

**可视化展示**:
- 词云图展示热点关键词
- 热度排行榜（柱状图）
- 情感分布饼图

### 2.4 机器学习选股（暂不做）

**预留功能接口**: 未来可扩展

## 3. 非功能需求

### 3.1 性能要求
- 页面首屏加载时间 < 3秒
- 实时数据刷新间隔 5-30秒
- 支持1000+数据点渲染流畅

### 3.2 数据源
- 股票行情数据: Tushare / AkShare / 东方财富API
- 财经帖子数据: 爬虫抓取雪球/股吧

### 3.3 技术栈（推荐）
- **前端**: React + ECharts / AntV G2
- **后端**: Python FastAPI
- **LLM**: OpenAI API / 通义千问 / 硅基流动
- **数据**: MySQL + Redis缓存
