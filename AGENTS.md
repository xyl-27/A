# A股智能选股分析平台 AI 开发指令

## 项目概述
使用 React + TypeScript + FastAPI 开发的 A股智能选股分析平台，集成大模型能力进行行业趋势分析、市场热点分析和投资决策辅助。

## 开发规范

### 技术栈要求
- **前端**: React 18 + TypeScript + ECharts + Ant Design
- **后端**: Python FastAPI + SQLAlchemy + Pydantic
- **数据库**: MySQL 8.0 + Redis 7.0
- **LLM**: OpenAI API (GPT-4) / 通义千问 / 硅基流动
- **部署**: Docker + Nginx

### 代码规范
- 使用 TypeScript，确保类型安全
- 使用 ESLint + Prettier 进行代码格式化
- API 接口使用 Pydantic 进行数据验证
- 前端使用 Zustand 进行状态管理
- 错误处理要有友好的用户提示

### 安全规范
- API Key 从环境变量读取，不要硬编码
- 实现 JWT 认证和 API 限流
- 防止 SQL 注入和 XSS 攻击
- 敏感数据要加密存储

## 功能模块开发要求

### 1. 行业趋势分析模块

#### 前端要求
- 实现可交互的行业树组件，支持展开/收起
- 使用 ECharts 绘制行业指数走势图
- 显示行业内个股涨跌幅分布
- 支持行业资金流向可视化

#### 后端要求
- 对接 Tushare Pro 和 AkShare 数据源
- 实现行业分类数据模型
- 提供行业指数和资金流向 API
- 实现数据缓存机制

#### LLM 集成要求
- 分析行业政策影响
- 生成行业趋势报告
- 提供投资建议摘要

### 2. 大盘星图可视化模块

#### 前端要求
- 使用 ECharts 实现气泡图可视化
- 气泡面积代表市值或成交额，可切换
- 气泡颜色代表涨跌幅，红色=上涨，绿色=下跌
- 支持鼠标悬停显示股票信息
- 支持双击跳转股票详情页
- 实现我的自选股高亮显示

#### 后端要求
- 提供星图数据 API，支持分页和筛选
- 实现大数据量的性能优化
- 支持实时数据更新

#### 性能要求
- 支持 5000+ 股票数据点渲染
- 页面首屏加载时间 < 3秒
- 使用虚拟化技术优化长列表

### 3. 市场热点分析模块

#### 爬虫开发要求
- 实现雪球热门帖子爬虫
- 实现东方财富股吧爬虫
- 遵守 robots.txt，控制请求频率
- 使用代理 IP 池防封禁
- 实现增量爬取和去重

#### LLM 分析要求
- 实现财经文本情感分析
- 提取热点关键词
- 生成热点话题总结
- 分析市场情绪倾向

#### 前端要求
- 使用词云图展示热点关键词
- 实现热点排行榜（柱状图）
- 显示情感分布（饼图）
- 支持热点话题详情查看

### 4. 数据服务模块

#### 数据库设计
- 实现股票基础信息表
- 实现股票行情数据表
- 实现行业数据表
- 实现热点话题表
- 建立合适的索引优化查询

#### API 设计
- 使用 RESTful API 设计规范
- 实现分页、筛选、排序功能
- 提供数据导出接口
- 实现 WebSocket 实时数据推送

## 技术挑战与解决方案

### 大数据量渲染
- **挑战**: 5000+ 股票数据点渲染
- **解决方案**: ECharts 大数据优化 + 虚拟化技术

### 实时数据更新
- **挑战**: 股票数据实时性要求高
- **解决方案**: WebSocket + Redis 缓存

### LLM 成本控制
- **挑战**: 大模型调用成本高
- **解决方案**: 智能缓存 + 调用频率控制

### 爬虫稳定性
- **挑战**: 财经网站反爬虫机制
- **解决方案**: 多源备份 + 频率控制 + 代理池

通过遵循以上开发规范，确保项目高质量、高性能地完成，为用户提供优秀的 A股智能选股分析体验。
